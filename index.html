Låt os börja med Element

Som bilden visar så är så kallade tags så som h1, h2, p, img mm. element medans allt innanför dem är den så kallade content

Klass

En generell klass som skrivs som class i taggen för att ge olika egenskaper till ett element som den inte skulle ha utan klassen

ID

ID används i en tag som en enstaka local klass, detta menar att du använder oftast ID klassen en endaste gång på ett enda element för att ha det så strukturerat som möjligt

HTTP och HTTPS

HTTP står för HyperText Transfer Protocol vilket betyder att den läser endast av och sedan visar vad datorn säger men det är allt den vissar ALLT så ifall du har information som är sårbar så ska du inte använda denna

Men HTTPS är samma förutom att S:et står för safe vilket betyder att den krypterar all inforamtion så att ingen annan förstår
